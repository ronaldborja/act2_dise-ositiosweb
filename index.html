¿Qué es el Machine Learning?

El Machine Learning es un subcampo de la Inteligencia Artificial que busca crear modelos a partir de conjuntos de datos usando algebra lineal y optimizarlos usando calculo.

Aplicaciones del Machine Learning

  • Reconocimiento de voz: Desarrollo de asistentes virtuales como Siri, Google Assitant, Alexa, entre otros...

  • Recomendaciones de productos:Personalización de la experiencia del usuario a partir de algoritmos de recomendación.

  • Detección de fraudes: Detección temprana de fraudes a partir de analisis en los patrones de gastos y comportamiento de los clientes.

  • Diagnóstico médico: Detección temprana de cancer, anomalías en imagenes, diabetes, enfermedades cardiacas, entre otros...

  • Vehiculos autonomos: Capacidad para reconocer señales de trafico, peatones y otros vehiculos.

Algoritmos de Machine Learning

Algoritmo Descripción
Regresión lineal Predice valores numéricos estableciendo relaciones lineales entre variables.
KNN Clasifica datos basados en vecinos más cercanos.
Decision Trees Divide datos en subconjuntos basados en decisiones específicas.
Neural Networks Aprende patrones complejos a través de conexiones neuronales interconectadas.
Suport Vector Machines Crea un hiperplano óptimo para clasificar datos.
Random Forest Promedia múltiples árboles de decisión para hacer predicciones.
